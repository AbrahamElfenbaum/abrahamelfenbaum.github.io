A selection of projects I've built, ranging from Unreal Engine C++ systems to smaller game and software experiments.

---

## [Project Iron Table](/projects/irontable/) — Unreal Engine 5 / C++ *(Active)*
A TTRPG simulator built from the ground up in Unreal Engine 5.7 and C++. Features physics-based dice rolling, a tabbed chat system with private channels, draggable UI panels, and persistent camera settings.

---

## [Random Name Generator](/projects/name-generator/) — C# *(2026)*
A C# machine learning project that analyzes a list of names and generates new ones that fit the same stylistic pattern. A personal exploration of ML applied to a game dev problem.

---

## [Frogdoku](/projects/frogdoku/) — Unreal Engine 5 / C++ *(Released October 2025)*
A cozy 3D grid-based puzzle game published on Steam by Digital Scorpion Interactive. I built the save, control binding, level timer, credits, and level select systems, as well as an in-game bug reporting tool.

---

## [Tales of Valoris: Swallow's Defenders](/projects/tales-of-valoris/) — Unity Engine / C# *(Released January 2025)*
A fantasy tower defense game published on Steam by Digital Scorpion Interactive. I built the Endless Mode systems, save/load system, bug report system, and Steam achievements, and served as QA Lead for the project.

---

## [Hit n' Bit](/projects/hit-n-bit/) — Unity Engine / C# *(Demo, 2024)*
A dark fantasy Metroidvania platformer by Digital Scorpion Interactive. I built the save system, controls, and scene manager, contributed to a range of other systems, and served as QA Lead for the demo.

---

## [White and Black](/projects/white-and-black/) — Unreal Engine 5 / Blueprints *(2022)*
A sandbox god simulator built in Unreal Engine 5 with Blueprints as my Full Sail capstone. I implemented the UI system, building placement, spell system, and audio across a four-person team.

---

## [Terrain Generator](/projects/terrain-generator/) — Unreal Engine 5 / Blueprints / C++ *(2022)*
A procedural terrain generator built in Unreal Engine using Perlin noise as an introductory assignment at Full Sail University.

---

## [SpellCraft](/projects/spellcraft/) — Unity Engine / C# *(2021)*
A Unity/C# midterm project at Full Sail University. I built the player controls, abilities, HUD, and character animations across a five-person team.

---

## [Game of Life](/projects/game-of-life/) — C# *(2021)*
A C# implementation of Conway's Game of Life, built as a Full Sail course assignment.

An upgrade/rework of the name generator I made. The current version reads and records patterns of letters and then generates a series of random letters based on the recorded results. I would like to see what a shift from pattern recognition to analysis would look like and how that would change the results of the output.

## Graphics Renderer — C++ / Vulkan / DirectX
I've made a graphics renderer once before in school, but it's been so long that I've decided to start from scratch and relearn the fundamentals using both Vulkan and DirectX. As a plus, by doing this in both pipelines, I'll be able to see how they compare in both their complexity in setting up and their performance.

## Custom Cipher & Encoder — C#
I love fantasy, and from that, I've been slowly working on a custom script. But I plan to take it further than that, I want to change the rules; only one way to make a word a plural, removing redundant and silent letters. From all of this, I want to make a tool that not only replaces letters with corresponding symbols, but recognizes the structure of the words and formats them to my new grammar rules accordingly.

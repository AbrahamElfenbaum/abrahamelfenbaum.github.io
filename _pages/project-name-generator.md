## Overview

A machine learning-based name generator that takes a list of names as input, analyzes their patterns, and produces new names that feel stylistically consistent with the input set. Feed it a list of fantasy names and it generates more fantasy names; feed it historical names and it generates historically plausible ones.

The project is a personal exploration of machine learning concepts applied to a problem that's directly useful in game development and worldbuilding. I work on it on and off as the inspiration strikes.

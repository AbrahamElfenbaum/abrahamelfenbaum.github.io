## Overview

SpellCraft was a midterm project developed collaboratively with fellow Full Sail students. My primary goal was to learn how to build player-controlled actions that felt responsive and satisfying to use.

---

## My Contributions

### Player Controls
Programmed how the player moves the character through the level.

### Player Abilities
Programmed the player's ability set — what each ability does, how often it can be used, and how it looks when activated.

### Player HUD
Built the in-game HUD, including a health bar that responds dynamically to damage and healing, and ability icons that disappear and refill to indicate cooldown state.

### Character Animations
Integrated character animations using Mixamo, covering all player actions including running, jumping, and death.
